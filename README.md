# EliTechWiz-V2

![EliTechWiz](https://upload.wikimedia.org/wikipedia/commons/e/ef/Youtube_logo.png)

## Introduction

EliTechWiz-V2 is a multi-device bot designed to enhance your WhatsApp experience with a variety of features. This bot is built using the Baileys library and offers a wide range of functionalities including media handling, group management, and more.

## Features

- **Media Handling**: Convert images, videos, and audio to different formats.
- **Group Management**: Manage group participants, send welcome/farewell messages, and more.
- **Automations**: Auto-reply, auto-download, and auto-sticker features.
- **Games**: Play games like Tic-Tac-Toe and Math Quiz.
- **Utilities**: Translate text, search the web, and more.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Eliahhango/elitechwiz-elitechwiz-v2.git
   cd EliTechWiz-V2
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the bot:
   ```bash
   npm start
   ```

## Configuration

Edit the `settings.js` file to configure the bot according to your needs. You can set the bot name, owner details, and other settings.

## Usage

- **Start the bot**: `npm start`
- **Stop the bot**: Press `Ctrl + C`
